# Data validation with Superstruct

A working demo for the companion blog post, [Data Validation with
Superstruct](https://dtjv.io/data-validation-with-superstruct).

## Install

Clone the repo and `cd` into this folder. Then...

```bash
$ yarn
```

## Run the code

```bash
$ yarn dev
```
