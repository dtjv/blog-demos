# The Generic Tree Data Structure

A working demo for the companion blog post,
[The Generic Tree Data Structure](https://dtjv.io/the-generic-tree-data-structure)

## Install

Clone the repo and `cd` into this folder. Then...

```bash
$ yarn install
```

## Run the code

```bash
$ yarn dev
```
